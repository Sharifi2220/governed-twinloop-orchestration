# Examples

This directory is reserved for short, public-safe illustrations of the governance pattern. It intentionally contains no operational example at this stage.

Future examples must be fictional or fully authorized, generic enough to avoid reconstructing a private workflow, and limited to the conceptual task-report-review cycle.

Examples must not include private prompts, customer-specific processes, real credentials or configuration, hidden scoring logic, production automation rules, deployment steps, or claims that this repository supplies working software.

Every proposed example requires review under the [public/private boundary](../docs/public-private-boundary.md) and [contribution policy](../governance/contribution-policy.md).

# Task card template

Task-ID: `PUBLIC-EXAMPLE-000`

## Goal

State one bounded outcome.

## Allowed files or resources

- List only the files, folders, or public sources required.

## Not allowed content or actions

- Identify sensitive content and actions outside the worker's authority.
- Do not include real secrets, private prompts, or operational control values.

## Expected output

Describe the artifact and evidence the worker must return.

## Stop conditions

- List missing access, uncertainty, boundary, or approval conditions that require handoff.

## Review criteria

- Define concise, observable checks for scope, quality, and safety.

## Report format

1. Executed steps
2. Created, changed, drafted, or skipped artifacts
3. Checks performed
4. Risks and uncertainties
5. Items requiring review
6. Recommendation
